 The carousing page has a generator on it which will allow you to get random carousing results for your players.
The combat page is an initiative tracker. It uses the data from the monster compendium page to allow you to pull in and roll HP and initiative for monsters, and it helps you keep track the state of combat all on one page.
The force page is a compendium of the Star Wars 5e force disciplines and abilities.
The gadgets page contains a list of all the available gadgets for the Engineer class, all the weapon modifications for the Trooper class, and all the gadgets usable by the Mercenary class.
The galaxy map page contains a force-directed graph of the Star Wars universe. On this page, you can learn things about the planets in the Star Wars universe such as the climate and inhabitants. In addition, you can use the tool presented to find the shortest path between certain planets.
The magical items page is a compendium of the magical items created for the Star Wars 5e conversion.
The monster manual page is a compendium of all of the Star Wars 5e monsters created for this conversion.
